Configuration & Parameters Registry

Единый источник всех настраиваемых параметров («магических чисел»). Цель — чтобы в коде не было хардкода, а у разработчика был один справочник: значение, диапазон, владелец, привязка к algo_version. Все значения provisional v0.9 до SME-валидации.

Правила

Параметры скоринга версионируются через algo_version (forward-only). Операционные (лимиты, TTL) — через конфиг окружения. Изменение скоринг-параметра = новая версия + запись в changelog/governance_log.

Для разработчика

Ни одно из этих чисел не хардкодится. Скоринг-параметры читаются по активной algo_version; операционные — из конфигурации окружения. Исторические Score пересчёту по новым значениям не подлежат (forward-only).
