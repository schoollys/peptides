UI Kit / Design System

Живой HTML: компоненты отрендерены реальным CSS со всеми состояниями (hover, active, disabled, error, loading, empty). Это каркас системы (токены + базовые компоненты), не финальный брендовый визуал — он за дизайнером. Палитра наследует визуальный язык документов проекта.
